Who deserve thanks

To deliver this webpage to you I have used several tools, and I would like to mention some of the most important ones here.

Operating Systems

  • OpenBSD Highly secure operating system
  • FreeBSD Great operating system with excellent performance
  • pfSense Hybrid firewall/router appliance operating system, based on FreeBSD and OpenBSD
  • FreeNAS Open Source Storage Platform based on FreeBSD
  • VMware ESXi Virtual machine hypervisor

Server Software

  • Varnish HTTP cache server
  • Connect Extensible HTTP server framework for Node.js
  • TinyDNS secure, reliable, small, fast, etc. DNS server
  • Nginx A high performance web server

Web development frameworks etc.

  • Node.js platform for easily building fast, scalable network applications with JavaScript
  • jQuery JavaScript library
  • Jekyll Simple blog aware static site generator
  • Twitter Bootstrap Frontend framework for faster and easier web development
  • LESS Dynamic stylesheet language

Other

  • favicon.cc favicon.ico generator
  • Color Scheme Designer Cool tool for picking color schemes
  • vim Highly configurable text editor
  • GitHub Free git code repository

 

And a big special thanks to Digitally Imported and Sky.fm for keeping me sane during the development.

While I do enjoy using proper IDEs most of the time, I wrote all the code (HTML, LESS, Node.js server, etc.) for these pages in vim, just for the kicks.
